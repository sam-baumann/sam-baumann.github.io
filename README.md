# Sam Baumann’s Personal Site

## Getting Started

```bash
pnpm install   # or npm ci
pnpm dev       # start the dev server
pnpm build     # build for production
pnpm preview   # preview the built site
```

## Site Structure:
- Landing Page: High level info, recent blogposts, social links
- Blog index: list of blog posts, including search and tag filtering
- /about: expanded "about", not quite resume but almost
- /now: what am I working on now? What music am I listening to? What's the last movie I watched?
